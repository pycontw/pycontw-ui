# pycontw-ui

This repository is a Storybook for PyCon TW website

## Set up Environment

1. Project setup

```
$ npm install
```

2. Compiles and hot-reloads for development

```
$ npm run serve
```

3. Compiles and minifies for production

```
$ npm run build
```

4. Lints and fixes files

```
$ npm run lint
```
